ğŸ†˜ TROUBLESHOOTING

PROBLEM: "adb devices" tidak menunjukkan handphone

SOLUSI 1: Cek USB Driver
- Install Android Studio (includes USB drivers)
- Atau install Google USB Driver

SOLUSI 2: Cek Kabel USB
- Ganti kabel USB (gunakan yang bagus)
- Coba port USB berbeda

SOLUSI 3: Cek Permission
- Di handphone: Settings > Developer options > USB Debugging
- Harus ON
- Accept permission popup jika ada

SOLUSI 4: Restart ADB
  adb kill-server
  adb start-server
  adb devices

SOLUSI 5: Clean & Reconnect
- Unplug USB kabel dari handphone
- Di handphone: Settings > Developer options > Toggle OFF USB Debugging
- Wait 10 seconds
- Toggle ON USB Debugging
- Plug USB kabel kembali
- Accept permission
